<h1>سياسة الخصوصية لتطبيق مدينة</h1>

<p><strong>تاريخ السريان:</strong> 16 فبراير 2026</p>

<p>نحترم خصوصيتك ونسعى لحماية بياناتك الشخصية. توضح هذه السياسة أنواع البيانات التي نجمعها، كيف نستخدمها، مع من نشاركها، وكيف يمكنك التحكم بها. باستخدامك لتطبيق مدينة، فإنك توافق على شروط هذه السياسة.</p>

<hr>

<h2>1. نطاق التطبيق والاسم السابق</h2>
<p>هذا المستند يصف سياسة الخصوصية لتطبيق <strong>مدينة</strong> (سابقًا: "غيمة العقارات") بعد التوسع في الخدمات لتشمل السوق (متجر)، نظام نشر بوستات مصنّفة، ومنصة الباحثين عن عمل بالإضافة إلى وظائف التواصل بين المستخدمين.</p>

<hr>

<h2>2. المعلومات التي نجمعها</h2>

<h3>أ. معلومات التسجيل والحساب</h3>
<ul>
  <li>بيانات الحساب: اسم المستخدم، البريد الإلكتروني، وصورة الملف الشخصي (إن زودت بها).</li>
  <li>طريقة التسجيل: تسجيل آمن عبر حساب Google أو عبر البريد الإلكتروني وكلمة المرور.</li>
</ul>

<h3>ب. المحتوى الذي تنشره</h3>
<ul>
  <li>المحتوى العام: بوستات، إعلانات بحث عن عمل، وصف المنتجات، صور أو فيديوهات ترفعها، تصنيفات تختارها عند النشر.</li>
  <li>تفاصيل المنتجات أو الخدمات: مواصفات المنتج، سعر مذكور من قبل البائع (إن وُضع)، ووسائل التواصل التي تختارها داخل التطبيق.</li>
</ul>

<h3>ج. البيانات الفنية والسلوكية</h3>
<ul>
  <li>بيانات الاستخدام: التفاعلات (مثل نشر، تعليق، تواصل مع بائع)، وإحصاءات الأداء لتطوير التطبيق.</li>
</ul>

<h3>د. بيانات اختيارية أو تقدمها طوعًا</h3>
<ul>
  <li>مؤهلات وخبرة المستخدم عند نشر "باحث عن عمل".</li>
  <li>عنوان العقار إذا قمت بإضافته صراحةً في وصف الإعلان.</li>
</ul>

<hr>

<h2>3. كيفية استخدامنا للبيانات</h2>
<ul>
  <li>لتشغيل الحسابات وتقديم خدمات التطبيق (عرض متجر، نشر بوستات، نشر إعلانات الباحث عن عمل، نظام الإشعارات).</li>
  <li>لتسهيل التواصل بين البائعين والمشترين داخل التطبيق.</li>
  <li>لإرسال إشعارات للمستخدمين عند وجود منشورات جديدة مهتمة بهم (مثل إشعار بوجود باحث عن عمل أو منتج جديد) — يمكنك التحكم في تفضيلات الإشعارات داخل إعدادات التطبيق.</li>
  <li>لتحسين أداء التطبيق وتجربة المستخدم، وإصلاح الأخطاء الفنية.</li>
  <li>للامتثال للالتزامات القانونية وحماية حقوقنا وحقوق المستخدمين.</li>
</ul>

<hr>

<h2>4. مشاركة البيانات مع أطراف ثالثة</h2>
<ul>
  <li>مقدمو الدخول عبر Google: عند تسجيل الدخول عبر Google، نتلقى المعلومات الأساسية التي توافق على مشاركتها عبر Google (مثل البريد الإلكتروني واسم الحساب) وفقًا لإعداداتك لدى مزودي الهوية.</li>
  <li>عدم مشاركة المحتوى العام: المحتوى الذي تنشره (بوستات، منتجات، إعلانات الباحث عن عمل) مرئي للمستخدمين الآخرين داخل التطبيق وفقًا لإعدادات الخصوصية ونطاق النشر الذي تختاره.</li>
  <li>لا نبيع بياناتك لجهات خارجية لأغراض تسويقية.</li>
</ul>

<hr>

<h2>5. المحتوى المقدم من المستخدمين</h2>
<ul>
  <li>كل منشور يتحمل الناشر مسؤوليته الكاملة — التطبيق مجرد منصة وسيط عرض/تواصل.</li>
  <li>لا يتحمّل التطبيق مسؤولية صحة أو اكتمال أو قانونية محتوى المنشورات أو الصفقات التي تتم خارج التطبيق بين طرفين.</li>
</ul>

<hr>

<h2>6. الدفعات والمعاملات المالية</h2>
<ul>
  <li>التطبيق لا يحتوي على نظام دفع داخلي؛ أي معاملة مالية تتم خارج التطبيق بين المستخدمين — نوصي بالحذر وعدم مشاركة بيانات بطاقات الدفع داخل أي دردشة أو منشور.</li>
</ul>

<hr>

<h2>7. كيف نحمي بياناتك</h2>
<ul>
  <li>نستخدم إجراءات أمان تقنية وإدارية مناسبة مثل تشفير الاتصالات (HTTPS) وتدابير داخلية للحد من الوصول غير المصرح به.</li>
  <li>مع ذلك، لا يمكن ضمان أمان البيانات بنسبة 100%؛ نوصي بعدم نشر معلومات حساسة للغاية داخل التطبيق.</li>
</ul>

<hr>

<h2>8. حفظ البيانات ومدة الاحتفاظ</h2>
<ul>
  <li>نحتفظ ببيانات الحساب والمحتوى الذي تنشره طالما حسابك نشط أو ما لم تطلب حذفه.</li>
  <li>بعد طلب حذف الحساب، قد نحتفظ بنسخ احتياطية مؤقتة لمدة تصل إلى 30 يومًا أو أطول إذا كان ذلك مطلوبًا للالتزامات القانونية أو لحماية حقوقنا.</li>
</ul>

<h2>9. حقوقك وطرق التحكم في بياناتك</h2>
<ul>
  <li>الاطّلاع: يمكنك الاطّلاع على بيانات حسابك ومحتوى منشوراتك.</li>
  <li>التعديل والحذف: يمكنك تعديل محتوى منشوراتك أو حذفها. لحذف الحساب بالكامل راجع قسم "حذف الحساب" أدناه.</li>
  <li>التحكم في الإشعارات: يمكنك تعطيل أو تقييد إشعارات التطبيق من داخل إعدادات التطبيق أو عبر إعدادات الهاتف.</li>
  <li>الاعتراض وحذف البيانات: يمكن تقديم طلبات لحذف البيانات الشخصية أو تقييد معالجتها عبر طرق التواصل الموضحة أدناه.</li>
</ul>
